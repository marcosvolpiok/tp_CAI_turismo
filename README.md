# Gestión de Turismo

Bienvenido al repositorio del proyecto "Gestión de Turismo", una aplicación de Windows Forms desarrollada con C#.

## Requisitos previos

Asegúrate de tener instalado lo siguiente antes de intentar ejecutar el proyecto:

- **Visual Studio:** Este proyecto está desarrollado en Visual Studio. Puedes descargar la versión gratuita Community de [Visual Studio](https://visualstudio.microsoft.com/).

## Instrucciones para abrir y ejecutar el proyecto

1. **Clonar el repositorio:**
   Abre tu terminal o Git Bash y ejecuta el siguiente comando para clonar este repositorio en tu máquina local:

   ```bash
   git clone https://github.com/marcosvolpiok/tp_CAI_turismo
Abrir el proyecto en Visual Studio:

Abre Visual Studio.
Selecciona "Archivo" > "Abrir" > "Proyecto/Solución".
Navega hasta la carpeta donde clonaste el repositorio y selecciona el archivo de solución (.sln) del proyecto.

Presiona F5 o selecciona "Depurar" > "Iniciar depuración" desde el menú. Esto compilará y ejecutará la aplicación.
¡Listo!

Tu aplicación de Gestión de Turismo debería ejecutarse ahora. Explora la funcionalidad según las características del proyecto.
